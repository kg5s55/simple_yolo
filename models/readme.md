## auto_pad函数说明

这个例子只考虑其中一个边，$x$是输入尺寸，$p$是padding尺寸，$k$是卷积核的尺寸，$d$是膨胀系数，$s$是步长，$y$ 是输出尺寸
$$
y = \frac{x+2p-(k-1)\cdot{d}+1}{s}+1
$$


模型构建参数列表

输入通道、输出通道，k,s

|   模块   |  参数1   | 参数2 | 参数3 |  参数4   |
| :------: | :------: | :---: | :---: | :------: |
|   Conv   |   c_in   | c_out |   k   |    s     |
|   C2f    |   c_in   | c_out |   n   | shortcut |
| Upsample | 默认顺序 |       |       |          |
|  Concat  |   dim    |       |       |          |
